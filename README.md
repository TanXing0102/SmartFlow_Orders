# SmartFlow Orders
SmartFlow Orders is an AI-powered shopping assistant that helps users quickly find products based on their intent, preferences, and budget. By leveraging natural language processing, the system transforms user queries into structured shopping plans and delivers intelligent recommendations through a chat-based interface.

---

## 🚀 Features
- AI intent extraction
- Chat-based interface
- Budget-aware recommendations

## ⚙️ Installation & Setup
1. Clone the Repository: git clone https://github.com/TanXing0102/SmartFlow_Orders

2. Backend Setup (Flask)
    cd Backend
    pip install flask
    pip install flask-cors
    pip install python-dotenv
    python app.py

Backend runs on: http://localhost:5000

3. Frontend Setup (React)
    cd Frontend/smartflow_orders-ui
    npm install
    npm start

Frontend runs on: http://localhost:3000
